# n8n Workflows & Automation 🤖

This is where I centralize my automation flows and the infrastructure I use to run everything locally in a professional way.

### Tools I use:
* **n8n:** Where I build the automation logic (emails, alerts, API integrations).
* **Docker & Docker Compose:** Used to spin up my PostgreSQL database and n8n environment without messing up my local OS.
* **MinIO:** Used as my local S3-compatible object storage.

### How I use this repo:
I export my n8n workflow `.json` files and save them here for version control and organization.
